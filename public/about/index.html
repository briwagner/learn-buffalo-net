About

Learn Buffalo wants to help you build great web applications. It's quick, simple and flexible. Use it to build RESTful applications, APIs and lots of other things.

Is it the best tool for every job? No, but we'll show some good uses for it.

Where can you learn more? The Buffalo docs site is a great start, and to explore the code, check out the project repos.
